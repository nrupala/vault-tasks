# ✅ Vault Tasks v1.1.0

**Private Task Management. Zero-Knowledge Security.**

Vault Tasks is a high-performance task management application designed for the privacy-conscious. It features hierarchical task lists, deadlines, and smart analytics, all secured with military-grade AES-256-GCM encryption entirely within your browser.

### 🌐 Live Demo
Visit: **[nrupala.github.io/vault-tasks](https://nrupala.github.io/vault-tasks/)**

## ✨ Key Features
- **Actionable Tasks**: Checklists with priority levels, flexible deadlines, and reminder notifications.
- **Unified Calendar**: A birds-eye view of your encrypted schedule.
- **Privacy Analytics**: Heatmaps and productivity trends processed locally.
- **Multi-Format Import**: Import tasks from **JSON**, **Text**, or **iCalendar (.ics)** files.

## 🔒 Security First
- **Local-Only Encryption**: Your data never touches a server.
- **Master Key Derivation**: PBKDF2 with 100,000 iterations.
- **Zero-Trust Architecture**: We have no access to your data, ever.

## 🛠️ Tech Stack
- **Core**: React 19, TypeScript, Vite
- **Storage**: IndexedDB (Dexie.js)
- **Encryption**: Web Crypto API (AES-GCM)
- **Mobile**: Capacitor (iOS/Android READY)
