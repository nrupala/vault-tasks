# @vault/tasks 📌

**Zero-Trust Encrypted Task Management**

A focused, independent module of the Vault Tracker ecosystem designed for high-performance task tracking with military-grade privacy.

## Features
- **AES-256-GCM Encryption**: All tasks are encrypted locally. Zero-trust architecture.
- **Focused Analytics**: Real-time insights into task completion rates and priority distribution.
- **Calendar Integration**: Visualize your deadlines and workload.
- **Media Attachments**: Securely attach images or documents to tasks.

## Getting Started
1. `npm install`
2. `npm run dev`

## Privacy Policy
Your data never leaves your device unencrypted. No tracking, no cloud syncing, 100% private.
