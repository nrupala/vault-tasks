# Dependencies

This project leverages the following core technologies:

- **React / Vite**: Frontend framework and build tool.
- **TailwindCSS**: Utility-first styling.
- **Framer Motion**: Premium micro-animations.
- **Lucide React**: Iconography.
- **@vault/core**: Shared cryptographic and persistence layer (Local).

## Cryptography
- **Web Crypto API**: High-performance, native AES-256-GCM encryption.
